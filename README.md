## 🚀 How to Run

### General Steps

1. Clone the repository

```bash id="repo2"
git clone https://github.com/your-username/java-repo.git
```

2. Navigate to any experiment folder

```bash id="repo3"
cd Exp-1 Calculator
```

3. Compile Java files

```bash id="repo4"
javac *.java
```

4. Run the program

```bash id="repo5"
java Main
```
