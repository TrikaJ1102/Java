# 🧮 Experiment 1: Java Calculator Application

## 📌 Objective

To develop a simple Java-based calculator application that performs basic arithmetic operations and demonstrates fundamental programming concepts such as:

* Input handling
* Conditional statements
* Methods
* Object-Oriented Programming basics

---

## 🧠 Problem Statement

Design and implement a calculator program in Java that can perform basic mathematical operations such as:

* Addition
* Subtraction
* Multiplication
* Division

The program should take user input, process the selected operation, and display the result.

---

## 🏗️ Project Structure

```
Exp-1 Calculator/
│
├── Main.java
├── Calculator.java
└── (other supporting files)
```

---

## ⚙️ Features

* ➕ Addition of numbers
* ➖ Subtraction of numbers
* ✖️ Multiplication of numbers
* ➗ Division of numbers
* 🧾 User input-based calculations
* ⚠️ Basic error handling (e.g., division by zero)

Basic Java calculators are commonly used to perform arithmetic operations efficiently and help beginners understand programming logic ([GitHub][1]).

---

## 🧠 Key Concepts Used

* ✅ Java Syntax & Structure
* ✅ Conditional Statements (`if-else`, `switch`)
* ✅ Methods and Function Calls
* ✅ User Input Handling (`Scanner`)
* ✅ Basic OOP Concepts

---

## 🔄 Working of the Program

1. User enters numbers and selects an operation
2. The program processes input using conditional logic
3. The corresponding arithmetic operation is executed
4. Result is displayed to the user

---

## 📊 Sample Output

```
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): *

Result: 50
```

## ⚠️ Important Observations

* Division by zero must be handled carefully
* Input validation is important for correct results
* Modular code (using methods) improves readability
